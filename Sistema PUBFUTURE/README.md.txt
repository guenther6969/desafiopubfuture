# DesafioPubFuture

Este é o repositorio para o teste Pratico da a PubFuture 2022, um programa de formação da Proway em conjunto com a Publica Tecnologia. Meu programa é feito interiamente em Python, apenas utilizando a biblioteca peewee(ORM) para editar o banco de dados, e a biblioteca PrettyTable para mostrar as informações no terminal. 

## Rodar o programa
Rodar esses comandos no terminal no diretorio do programa
```bash
pip install PrettyTable
pip install peewee
python3 model.py
python3 main.py
```

#DataBase
Durante a criação da DataBase fiz exatamento como pedido no teste, utilizando uma table para Contas, Receitas e Despesas, criei elas utilizando uma ORM chamada peewee, similar como SQLAlchemy e Hibernate(Java) para representar tabelas da database em classes em python, não consegui utilizar APIs pois utilizei somente o terminal.
